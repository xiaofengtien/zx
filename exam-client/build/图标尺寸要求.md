# Electron 图标尺寸要求详解

## 默认 Electron 图标尺寸

Electron 默认图标（原子图标）的实际尺寸：
- **默认图标尺寸**：通常是 **512x512 像素**（正方形）
- **图标格式**：PNG（开发环境）或 ICNS/ICO（生产环境）

## 推荐的图标尺寸要求

### 开发环境（使用 PNG）

**最小要求**：
- **推荐尺寸**：**512x512 像素**（正方形）
- **格式**：PNG（支持透明背景）
- **形状**：正方形（1:1 比例）

**最佳实践**：
- **源文件尺寸**：**1024x1024 像素**（正方形）
- 这样可以确保在生成不同尺寸时保持清晰度

### 生产环境

#### macOS（ICNS 格式）
需要包含以下所有尺寸：
- 16x16 像素（标准）
- 32x32 像素（@2x，即 16x16@2x）
- 32x32 像素（标准）
- 64x64 像素（@2x，即 32x32@2x）
- 128x128 像素（标准）
- 256x256 像素（@2x，即 128x128@2x）
- 256x256 像素（标准）
- 512x512 像素（@2x，即 256x256@2x）
- 512x512 像素（标准）
- 1024x1024 像素（@2x，即 512x512@2x）

#### Windows（ICO 格式）
建议包含以下尺寸：
- 16x16 像素
- 20x20 像素（125% DPI）
- 24x24 像素（150% DPI）
- 32x32 像素
- 40x40 像素（125% DPI）
- 48x48 像素（150% DPI）
- 64x64 像素（200% DPI）
- 256x256 像素

## 如何调整 logo.png

### 1. 检查当前 logo.png 尺寸

```bash
# 查看当前文件信息
file exam-client/src/renderer/assets/images/logo.png

# 或使用 sips（macOS）
sips -g pixelWidth -g pixelHeight exam-client/src/renderer/assets/images/logo.png
```

### 2. 调整到推荐尺寸（1024x1024）

#### 方法一：使用 sips（macOS 自带）

```bash
cd exam-client/src/renderer/assets/images

# 调整到 1024x1024（保持比例，添加透明背景）
sips -z 1024 1024 logo.png --out logo_1024.png

# 或者强制调整为正方形（可能会裁剪）
sips -Z 1024 logo.png --out logo_1024.png
```

#### 方法二：使用 ImageMagick

```bash
# 安装 ImageMagick（如果未安装）
# macOS: brew install imagemagick

# 调整到 1024x1024（保持比例，添加透明背景）
convert logo.png -background transparent -gravity center -extent 1024x1024 logo_1024.png

# 或者直接缩放（可能会变形）
convert logo.png -resize 1024x1024 logo_1024.png
```

#### 方法三：使用在线工具

1. 访问：https://www.iloveimg.com/resize-image
2. 上传 `logo.png`
3. 选择 "自定义尺寸"
4. 设置宽度和高度为 **1024x1024**
5. 选择 "保持宽高比" 或 "强制尺寸"
6. 下载调整后的图片

#### 方法四：使用图像编辑软件

**Photoshop / GIMP / Figma**：
1. 打开 `logo.png`
2. 创建 1024x1024 的新画布（透明背景）
3. 将 logo 居中放置
4. 调整大小，确保在 1024x1024 画布内
5. 导出为 PNG（保持透明背景）

### 3. 图标设计建议

#### 尺寸和形状
- ✅ **必须是正方形**（1:1 比例）
- ✅ **推荐 1024x1024 像素**（源文件）
- ✅ **最小 512x512 像素**（开发环境）
- ❌ 避免使用矩形或非正方形图标

#### 内容设计
- ✅ **简洁明了**：在小尺寸（16x16）下也能识别
- ✅ **避免过多细节**：细节在小尺寸下会丢失
- ✅ **使用高对比度**：确保在不同背景下都清晰可见
- ✅ **透明背景**：PNG 格式支持透明，更灵活

#### 安全区域
- 在 1024x1024 的图标中，**核心内容应该在中心 800x800 像素区域**
- 边缘 112 像素（每边）留作安全边距
- 这样可以避免在 macOS 圆角裁剪时内容被切掉

### 4. 验证调整后的图标

```bash
# 检查尺寸
sips -g pixelWidth -g pixelHeight exam-client/src/renderer/assets/images/logo.png

# 应该显示：
# pixelWidth: 1024
# pixelHeight: 1024
```

## 快速调整脚本（macOS）

创建一个脚本来自动调整图标：

```bash
#!/bin/bash
# 保存为 adjust_logo.sh

LOGO_PATH="exam-client/src/renderer/assets/images/logo.png"
TEMP_PATH="exam-client/src/renderer/assets/images/logo_temp.png"

echo "检查当前 logo.png 尺寸..."
sips -g pixelWidth -g pixelHeight "$LOGO_PATH"

echo "调整到 1024x1024..."
sips -z 1024 1024 "$LOGO_PATH" --out "$TEMP_PATH"

echo "替换原文件..."
mv "$TEMP_PATH" "$LOGO_PATH"

echo "✅ 完成！新尺寸："
sips -g pixelWidth -g pixelHeight "$LOGO_PATH"
```

使用方法：
```bash
chmod +x adjust_logo.sh
./adjust_logo.sh
```

## 总结

**开发环境推荐**：
- 尺寸：**1024x1024 像素**（正方形）
- 格式：PNG（透明背景）
- 位置：`exam-client/src/renderer/assets/images/logo.png`

**生产环境**：
- macOS：从 1024x1024 PNG 生成 ICNS（包含所有尺寸）
- Windows：从 1024x1024 PNG 生成 ICO（包含所有尺寸）

**关键点**：
1. ✅ 必须是正方形（1:1）
2. ✅ 推荐 1024x1024 像素
3. ✅ 透明背景 PNG
4. ✅ 简洁设计，小尺寸下可识别
