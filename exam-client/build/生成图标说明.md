# 生成应用图标说明

## 当前状态

✅ **已完成的配置**：
- `package.json` 中已配置 `productName: "择学考试端"`
- `main.js` 中已设置 `app.setName('择学考试端')`
- Windows 图标路径：`build/icon.ico`
- macOS 图标路径：`build/icon.icns`

## 需要生成的图标文件

### 1. Windows图标 (icon.ico)
- 位置：`exam-client/build/icon.ico`
- 要求：包含多个尺寸（16x16, 32x32, 48x48, 256x256）
- 源文件：可以使用 `build/icon.png`（已从logo.png复制）

### 2. macOS图标 (icon.icns)
- 位置：`exam-client/build/icon.icns`
- 要求：包含多个尺寸（16x16, 32x32, 128x128, 256x256, 512x512, 1024x1024）
- 源文件：可以使用 `build/icon.png`（已从logo.png复制）

## 快速生成方法

### 方法一：使用在线工具（推荐）

1. **ICO转换（Windows）**：
   - 访问：https://convertio.co/zh/png-ico/
   - 上传 `build/icon.png`
   - 下载生成的 `icon.ico` 到 `build/` 目录

2. **ICNS转换（macOS）**：
   - 访问：https://cloudconvert.com/png-to-icns
   - 上传 `build/icon.png`
   - 下载生成的 `icon.icns` 到 `build/` 目录

### 方法二：使用命令行工具（macOS）

#### 生成ICNS（macOS自带工具）：
```bash
cd exam-client/build

# 1. 创建iconset目录
mkdir icon.iconset

# 2. 生成不同尺寸（需要1024x1024的源图，如果icon.png不够大，需要先放大）
# 如果icon.png是1024x1024，直接使用；否则需要先调整大小
sips -z 16 16     icon.png --out icon.iconset/icon_16x16.png
sips -z 32 32     icon.png --out icon.iconset/icon_16x16@2x.png
sips -z 32 32     icon.png --out icon.iconset/icon_32x32.png
sips -z 64 64     icon.png --out icon.iconset/icon_32x32@2x.png
sips -z 128 128   icon.png --out icon.iconset/icon_128x128.png
sips -z 256 256   icon.png --out icon.iconset/icon_128x128@2x.png
sips -z 256 256   icon.png --out icon.iconset/icon_256x256.png
sips -z 512 512   icon.png --out icon.iconset/icon_256x256@2x.png
sips -z 512 512   icon.png --out icon.iconset/icon_512x512.png
sips -z 1024 1024 icon.png --out icon.iconset/icon_512x512@2x.png

# 3. 生成icns文件
iconutil -c icns icon.iconset -o icon.icns

# 4. 清理临时文件
rm -rf icon.iconset
```

#### 生成ICO（需要ImageMagick）：
```bash
# 安装ImageMagick（如果未安装）
# macOS: brew install imagemagick
# Windows: 下载安装包

# 生成ICO
convert icon.png -define icon:auto-resize=256,128,64,48,32,16 icon.ico
```

## 验证图标文件

生成后，确认以下文件存在：
- ✅ `exam-client/build/icon.ico` (Windows)
- ✅ `exam-client/build/icon.icns` (macOS)

## 打包测试

生成图标后，运行打包命令：
```bash
# Windows
npm run pack:win

# macOS
npm run pack:mac
```

打包完成后，检查：
- Windows: `.exe` 文件图标应为自定义图标
- macOS: `.dmg` 和应用图标应为自定义图标
- 应用名称应显示为"择学考试端"（不是"Electron"）

## 注意事项

1. **源图片要求**：
   - 建议使用1024x1024或更大的正方形PNG
   - 背景应为透明（PNG格式）
   - 图标内容应清晰，在小尺寸下也能识别

2. **如果logo.png太小**：
   - 可以使用图像编辑软件（如Photoshop、GIMP）放大
   - 或使用在线工具：https://www.iloveimg.com/resize-image

3. **图标设计建议**：
   - 使用简洁的设计
   - 避免过多细节
   - 确保在16x16尺寸下也能识别
