# 试卷功能增强设计方案

## 一、需求概述

基于提供的图片和需求，需要实现以下功能增强：

1. **练习次数限制**：限制学员练习次数，以提交答案为一次
2. **试听功能**：提供试听文案、图片示例、音频示例
3. **注意事项**：试卷注意事项配置和显示
4. **试卷下载成功提示**：优化提示信息
5. **题目音频支持**：题目可以包含音频
6. **简化试卷字段**：取消年月省必填，隐藏这些字段，只保留自定义试卷名称+类型+AB卷
7. **AB卷设置**：支持A卷和B卷的独立配置
8. **答题讲解**：答题后提供讲解（含音频）
9. **大题区分**：试卷支持大题（如第一节、第二节），每个大题有题目数量和分数
10. **A卷B卷流程**：A卷提交成功后播放中场音频，然后才能做B卷，B卷完成后才算整个试卷完成

### 2.2 PaperVolume表（试卷卷别表）- 新建

```sql
CREATE TABLE paper_volume (
  id INT PRIMARY KEY AUTO_INCREMENT,
  paper_id INT NOT NULL COMMENT '试卷ID',
  volume_code VARCHAR(10) NOT NULL COMMENT '卷别代码（A、B、C等）',
  volume_name VARCHAR(100) NOT NULL COMMENT '卷别名称（如：试卷A、试卷B）',
  volume_order INT NOT NULL COMMENT '卷别顺序（1,2,3...）',
  volume_audio_url VARCHAR(500) COMMENT '卷别名称音频URL（播放试卷名称）',
  volume_audio_path VARCHAR(500) COMMENT '卷别名称音频本地路径',
  volume_audio_duration INT COMMENT '卷别名称音频时长（秒）',
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  INDEX idx_paper_volume (paper_id, volume_order),
  UNIQUE KEY uk_paper_volume (paper_id, volume_code)
) COMMENT='试卷卷别表';
```

### 2.3 PaperSection表（大题表）- 新建

```sql
CREATE TABLE paper_section (

  paper_id INT NOT NULL COMMENT '试卷ID',

  total_score DECIMAL(10,2) NOT NULL DEFAULT 0 COMMENT '总分（自动计算）',
  score_per_question DECIMAL(10,2) COMMENT '每题分数（如果统一）',

  instruction_audio_path VARCHAR(500) COMMENT '大题说明音频本地路径',
```

## 三、功能流程设计

### 3.1 试卷选择流程

```
1. 学员登录
   ↓
2. 显示可用试卷列表
   - 显示试卷名称、类型
   - 显示练习次数限制和已练习次数
   - 如果已达到练习次数限制，禁用"开始练习"按钮
   ↓
3. 学员选择试卷
   ↓
4. 检查练习次数
   - 如果已达到限制，提示"已达到练习次数限制"
   - 如果未达到限制，继续
   ↓
5. 进入试听页面（如果启用）
```

### 3.2 试听流程

```
1. 显示试听页面
   - 显示试听提示文案
   - 显示图片示例（耳机佩戴示意图）
   - 提供试听音频播放按钮
   ↓
2. 学员点击"试听"按钮
   - 播放试听音频
   - 显示音量调节提示
   ↓
3. 学员确认
   - 点击"清晰"按钮 → 进入注意事项页面
   - 点击"不清晰"按钮 → 提示联系监考老师
```

### 3.3 注意事项流程

```
1. 显示注意事项页面
   - 显示试卷注意事项（支持富文本）
   - 显示"我已阅读"确认按钮
   ↓
2. 学员确认
   - 点击"我已阅读" → 开始下载试卷
```

### 3.4 试卷下载流程

```
1. 显示"正在下载试卷..."提示
   ↓
2. 下载ZIP包
   - 显示下载进度
   ↓
3. 解压ZIP包
   - 提取manifest.json
   - 提取questions.json
   - 提取媒体文件（音频、图片）
   ↓
4. 显示"试卷下载成功，测试准备就绪"
   - 提示"请安静等待测试开始，不要再动耳机、键盘、鼠标等设备"
   ↓
5. 等待试卷开始
```

### 3.5 答题流程（多卷：A、B、C等）

#### 3.5.1 多卷答题流程（通用）

```
1. 显示当前卷标题和说明
   - 播放卷别名称音频（如："试卷A"）
   - 等待1-2秒后自动显示答题讲解（如果有）
   - 显示卷别说明（如果有）
   ↓
2. 显示大题列表
   - 显示第一节、第二节等（按后台配置的顺序）
   - 每个大题显示题目数量和分数
   ↓
3. 开始第一节
   - 播放第一节说明音频（如果有）
   - 显示第一节题目（按后台配置的顺序，可拖拽调整）
   - 播放题目音频（如果有）
   - 学员答题
   ↓
4. 继续后续大题
   - 重复步骤3
   ↓
5. 当前卷答题完成
   - 显示"X卷作答完成，答案上传成功！"
   - 检查是否有下一卷
   ↓
6. 如果有下一卷
   - 播放中场音频（不支持跳过）
   - 显示中场提示文案
   - 等待中场音频播放完成
   - 进入下一卷（重复步骤1-5）
   ↓
7. 如果所有卷都完成
   - 显示"整个试卷答题完成"
   - 提交整个试卷
   - 更新练习次数（+1，按试卷提交次数计算）
   - 更新答题记录状态为"已完成"
```

### 3.6 答题讲解流程

```
1. 播放完试卷名称音频（如："试卷A"）
   ↓
2. 等待1-2秒（可配置延迟时间）
   ↓
3. 自动显示答题讲解
   - 显示讲解文字
   - 播放讲解音频（如果有）
   ↓
4. 学员可以关闭讲解，继续答题
```

## 五、后端API设计

### 5.1 试卷管理API

#### 5.1.1 创建/更新试卷
```
POST /paper/add
POST /paper/edit

请求体包含：
- paperName（试卷名称）
- paperType（试卷类型）
- practiceLimit（练习次数限制，按试卷提交次数计算）
- trialListenEnabled（是否启用试听）
- trialListenText（试听提示文案）
- trialListenImageUrl（试听图片URL）
- trialListenAudioUrl（试听音频URL）
- notes（注意事项）
- volumes（卷别列表，如：[{"volumeCode":"A","volumeName":"试卷A"},{"volumeCode":"B","volumeName":"试卷B"}]）
- intermissions（中场配置列表，如：[{"fromVolume":"A","toVolume":"B","audioUrl":"...","text":"..."}]）
- sections（大题列表，按卷别和大题顺序组织）
```

#### 5.1.2 查询试卷列表
```
POST /paper/list

返回字段：
- id
- paperName
- paperType
- practiceLimit（练习次数限制）
- volumes（卷别列表）
- 不返回year, month, province（隐藏字段）
```

#### 5.1.3 查询试卷详情
```
GET /paper/{id}

返回完整试卷信息，包括：
- 基本信息
- 大题列表（sections）
- 题目列表（questions）
```

### 5.2 卷别管理API

#### 5.2.1 创建/更新卷别
```
POST /paper/volume/add
POST /paper/volume/edit

请求体包含：
- paperId
- volumeCode（A、B、C等）
- volumeName（卷别名称）
- volumeOrder（顺序）
- volumeAudioUrl（卷别名称音频URL）
```

#### 5.2.2 查询卷别列表
```
GET /paper/{paperId}/volumes

返回指定试卷的所有卷别列表
```

### 7.1 生成流程（使用新的表结构）

```
1. 获取试卷基本信息
   - 从 paper 表获取试卷基本信息
   - 从 paper_volume 表获取卷别列表（A、B、C等）
   - 从 paper_section 表获取大题列表（按卷别分组）
   - 从 paper_intermission 表获取中场配置
   - 从 paper_question 表获取题目关联（按 section_id 和 section_order 排序）
   ↓
2. 构建 manifest.json
   - 试卷基本信息
   - 卷别列表（包含卷别媒体文件，从 question_media 表查询 media_type=7）
   - 大题列表（包含大题说明媒体文件，从 question_media 表查询 media_type=8）
   - 中场配置（包含中场媒体文件，从 question_media 表查询 media_type=9）
   - 试听配置（包含试听媒体文件，从 question_media 表查询 media_type=10,11）
   ↓
3. 获取题目列表
   - 从 paper_question 表获取题目ID列表（按 section_id 和 section_order 排序）
   - 从 question 表获取题目详细信息
   ↓
4. 构建 questions.json
   - 题目基本信息
   - 题目媒体文件（从 question_media 表查询 media_type=4）
   - 讲解信息（explanationEnabled, explanationText, explanationDelaySeconds）
   - 讲解媒体文件（从 question_media 表查询 media_type=5）
   - 选项列表（包含选项媒体文件，从 question_media 表查询 media_type=2）
   ↓
5. 下载并打包媒体文件
   - 从 question_media 表查询所有相关媒体文件：
     * 试听媒体（media_type=10,11，关联到 paper）
     * 卷别名称音频（media_type=7，关联到 paper_volume）
     * 大题说明音频（media_type=8，关联到 paper_section）
     * 中场音频（media_type=9，关联到 paper_intermission）
     * 题目音频（media_type=4，关联到 question）
     * 讲解音频（media_type=5，关联到 question）
     * 选项音频（media_type=2，关联到 question_answer）
     * 题目内容媒体（media_type=1，关联到 question）
   - 从OSS下载所有媒体文件
   - 按目录结构组织到ZIP包中
   ↓
6. 生成ZIP包
   - 写入 manifest.json
   - 写入 questions.json
   - 写入所有媒体文件
   - 计算hash和大小
   ↓
7. 上传到OSS并更新数据库
   - 上传ZIP包到OSS
   - 更新 paper 表的 package_hash, package_size, version 等字段
```

```java
private Map<String, Object> buildManifest(Paper paper, 
                                         List<PaperVolume> volumes,
                                         List<PaperSection> sections,
                                         List<PaperIntermission> intermissions,
                                         List<PaperQuestion> paperQuestions) {
    Map<String, Object> manifest = new HashMap<>();
    
    // 1. 基本信息
    manifest.put("paperId", paper.getId());
    manifest.put("paperCode", paper.getPaperCode());
    manifest.put("paperName", paper.getPaperName());
    manifest.put("paperType", paper.getPaperType());
    manifest.put("practiceLimit", paper.getPracticeLimit());
    manifest.put("notes", paper.getNotes());
    
    // 2. 试听配置（从 question_media 表查询 media_type=10,11）
    if (paper.getTrialListenEnabled()) {
        List<QuestionMedia> trialMedia = questionMediaBiz.list(
            new LambdaQueryWrapper<QuestionMedia>()
                .eq(QuestionMedia::getPaperId, paper.getId())
                .in(QuestionMedia::getMediaType, Arrays.asList(10, 11))
        );
        manifest.put("trialListenEnabled", true);
        manifest.put("trialListenText", paper.getTrialListenText());
        manifest.put("trialListenMedia", buildMediaList(trialMedia, "trial_listen/"));
    }
    
    // 3. 卷别列表（从 paper_volume 表查询，包含卷别媒体）
    List<Map<String, Object>> volumesData = new ArrayList<>();
    for (PaperVolume volume : volumes) {
        Map<String, Object> volumeData = new HashMap<>();
        volumeData.put("volumeCode", volume.getVolumeCode());
        volumeData.put("volumeName", volume.getVolumeName());
        volumeData.put("volumeOrder", volume.getVolumeOrder());
        
        // 查询卷别名称音频（media_type=7）
        List<QuestionMedia> volumeMedia = questionMediaBiz.list(
            new LambdaQueryWrapper<QuestionMedia>()
                .eq(QuestionMedia::getVolumeId, volume.getId())
                .eq(QuestionMedia::getMediaType, 7)
        );
        volumeData.put("volumeMedia", buildMediaList(volumeMedia, "volumes/"));
        volumesData.add(volumeData);
    }
    manifest.put("volumes", volumesData);
    
    // 4. 中场配置（从 paper_intermission 表查询，包含中场媒体）
    List<Map<String, Object>> intermissionsData = new ArrayList<>();
    for (PaperIntermission intermission : intermissions) {
        Map<String, Object> intermissionData = new HashMap<>();
        intermissionData.put("fromVolume", intermission.getFromVolume());
        intermissionData.put("toVolume", intermission.getToVolume());
        intermissionData.put("intermissionText", intermission.getIntermissionText());
        intermissionData.put("canSkip", intermission.getCanSkip() == 1);
        
        // 查询中场音频（media_type=9）
        List<QuestionMedia> intermissionMedia = questionMediaBiz.list(
            new LambdaQueryWrapper<QuestionMedia>()
                .eq(QuestionMedia::getIntermissionId, intermission.getId())
                .eq(QuestionMedia::getMediaType, 9)
        );
        intermissionData.put("intermissionMedia", buildMediaList(intermissionMedia, "intermission/"));
        intermissionsData.add(intermissionData);
    }
    manifest.put("intermissions", intermissionsData);
    
    // 5. 大题列表（从 paper_section 表查询，包含大题说明媒体）
    List<Map<String, Object>> sectionsData = new ArrayList<>();
    for (PaperSection section : sections) {
        Map<String, Object> sectionData = new HashMap<>();
        sectionData.put("id", section.getId());
        sectionData.put("volumeCode", section.getVolumeCode());
        sectionData.put("sectionName", section.getSectionName());
        sectionData.put("sectionOrder", section.getSectionOrder());
        sectionData.put("questionCount", section.getQuestionCount());
        sectionData.put("totalScore", section.getTotalScore());
        sectionData.put("instructionText", section.getInstructionText());
        
        // 查询大题说明音频（media_type=8）
        List<QuestionMedia> sectionMedia = questionMediaBiz.list(
            new LambdaQueryWrapper<QuestionMedia>()
                .eq(QuestionMedia::getSectionId, section.getId())
                .eq(QuestionMedia::getMediaType, 8)
        );
        sectionData.put("instructionMedia", buildMediaList(sectionMedia, "sections/"));
        
        // 查询大题下的题目ID列表（按 section_order 排序）
        List<Integer> questionIds = paperQuestions.stream()
            .filter(pq -> pq.getSectionId().equals(section.getId()))
            .sorted(Comparator.comparing(PaperQuestion::getSectionOrder))
            .map(PaperQuestion::getQuestionId)
            .collect(Collectors.toList());
        sectionData.put("questions", questionIds);
        
        sectionsData.add(sectionData);
    }
    manifest.put("sections", sectionsData);
    
    manifest.put("version", paper.getVersion() != null ? paper.getVersion() : 0);
    manifest.put("createTime", paper.getCreateTime() != null ? paper.getCreateTime().getTime() : null);
    manifest.put("updateTime", paper.getUpdateTime() != null ? paper.getUpdateTime().getTime() : null);
    
    return manifest;
}

 * 构建媒体文件列表（用于JSON序列化）

private List<Map<String, Object>> buildMediaList(List<QuestionMedia> mediaList, String basePath) {
    return mediaList.stream().map(media -> {
        Map<String, Object> mediaData = new HashMap<>();
        mediaData.put("mediaType", media.getMediaType());
        mediaData.put("mediaPath", basePath + media.getMediaName()); // 构建ZIP中的相对路径
        mediaData.put("mediaFormat", media.getMediaFormat());
        if (media.getMediaDuration() != null) {
            mediaData.put("mediaDuration", media.getMediaDuration());
        }
        return mediaData;
    }).collect(Collectors.toList());
}
```

```java
private List<Map<String, Object>> buildQuestionsData(List<Integer> questionIds, 
                                                      List<PaperQuestion> paperQuestions) throws ServiceException {
    List<Map<String, Object>> questionsData = new ArrayList<>();
    
    // 构建题目ID到sectionId和sectionOrder的映射
    Map<Integer, PaperQuestion> questionMap = paperQuestions.stream()
        .collect(Collectors.toMap(PaperQuestion::getQuestionId, pq -> pq));
    
    for (Integer questionId : questionIds) {
        // 获取题目详情
        QuestionInfoDTO questionDTO = questionService.getQuestion(new QuestionIdBO(questionId));
        if (questionDTO == null) {
            continue;
        }
        
        PaperQuestion paperQuestion = questionMap.get(questionId);
        
        Map<String, Object> questionData = new HashMap<>();
        questionData.put("id", questionDTO.getId());
        questionData.put("sectionId", paperQuestion != null ? paperQuestion.getSectionId() : null);
        questionData.put("sectionOrder", paperQuestion != null ? paperQuestion.getSectionOrder() : null);
        questionData.put("title", questionDTO.getTitle());
        questionData.put("type", questionDTO.getType());
        questionData.put("explanationEnabled", questionDTO.getExplanationEnabled());
        questionData.put("explanationText", questionDTO.getExplanationText());
        questionData.put("explanationDelaySeconds", questionDTO.getExplanationDelaySeconds());
        
        // 查询题目媒体文件（media_type=4,5,6）
        List<QuestionMedia> questionMedia = questionMediaBiz.list(
            new LambdaQueryWrapper<QuestionMedia>()
                .eq(QuestionMedia::getQuestionId, questionId)
                .in(QuestionMedia::getMediaType, Arrays.asList(4, 5, 6))
        );
        
        // 分别处理题目音频（media_type=4）和讲解音频（media_type=5）
        List<Map<String, Object>> mediaList = new ArrayList<>();
        for (QuestionMedia media : questionMedia) {
            Map<String, Object> mediaData = new HashMap<>();
            mediaData.put("mediaType", media.getMediaType());
            
            // 根据media_type确定路径
            String mediaPath;
            if (media.getMediaType() == 4) {
                mediaPath = "questions/" + media.getMediaName();
            } else if (media.getMediaType() == 5) {
                mediaPath = "explanations/" + media.getMediaName();
            } else {
                mediaPath = "questions/" + media.getMediaName();
            }
            
            mediaData.put("mediaPath", mediaPath);
            mediaData.put("mediaFormat", media.getMediaFormat());
            if (media.getMediaDuration() != null) {
                mediaData.put("mediaDuration", media.getMediaDuration());
            }
            mediaList.add(mediaData);
        }
        questionData.put("media", mediaList);
        
        // 获取答案选项（包含选项媒体 media_type=2）
        if (questionDTO.getAnswers() != null) {
            List<Map<String, Object>> answersData = new ArrayList<>();
            for (QuestionAnswerDTO answer : questionDTO.getAnswers()) {
                Map<String, Object> answerData = new HashMap<>();
                answerData.put("id", answer.getId());
                answerData.put("text", answer.getOptionContent());
                
                // 查询选项媒体文件（media_type=2）
                List<QuestionMedia> optionMedia = questionMediaBiz.list(
                    new LambdaQueryWrapper<QuestionMedia>()
                        .eq(QuestionMedia::getOptionId, answer.getId())
                        .eq(QuestionMedia::getMediaType, 2)
                );
                
                List<Map<String, Object>> optionMediaList = optionMedia.stream().map(media -> {
                    Map<String, Object> mediaData = new HashMap<>();
                    mediaData.put("mediaType", media.getMediaType());
                    mediaData.put("mediaPath", "options/" + media.getMediaName());
                    mediaData.put("mediaFormat", media.getMediaFormat());
                    if (media.getMediaDuration() != null) {
                        mediaData.put("mediaDuration", media.getMediaDuration());
                    }
                    return mediaData;
                }).collect(Collectors.toList());
                
                answerData.put("media", optionMediaList);
                answersData.add(answerData);
            }
            questionData.put("answers", answersData);
        }
        
        questionsData.add(questionData);
    }
    
    return questionsData;
}
```

```java
private void downloadAndPackMediaFiles(ZipOutputStream zos, Paper paper, 
                                       List<PaperVolume> volumes,
                                       List<PaperSection> sections,
                                       List<PaperIntermission> intermissions,
                                       List<Integer> questionIds) throws Exception {
    
    // 1. 试听媒体（media_type=10,11，关联到 paper）
    List<QuestionMedia> trialMedia = questionMediaBiz.list(
        new LambdaQueryWrapper<QuestionMedia>()
            .eq(QuestionMedia::getPaperId, paper.getId())
            .in(QuestionMedia::getMediaType, Arrays.asList(10, 11))
    );
    packMediaFiles(zos, trialMedia, "trial_listen/");
    
    // 2. 卷别名称音频（media_type=7，关联到 paper_volume）
    for (PaperVolume volume : volumes) {
        List<QuestionMedia> volumeMedia = questionMediaBiz.list(
            new LambdaQueryWrapper<QuestionMedia>()
                .eq(QuestionMedia::getVolumeId, volume.getId())
                .eq(QuestionMedia::getMediaType, 7)
        );
        packMediaFiles(zos, volumeMedia, "volumes/");
    }
    
    // 3. 大题说明音频（media_type=8，关联到 paper_section）
    for (PaperSection section : sections) {
        List<QuestionMedia> sectionMedia = questionMediaBiz.list(
            new LambdaQueryWrapper<QuestionMedia>()
                .eq(QuestionMedia::getSectionId, section.getId())
                .eq(QuestionMedia::getMediaType, 8)
        );
        packMediaFiles(zos, sectionMedia, "sections/");
    }
    
    // 4. 中场音频（media_type=9，关联到 paper_intermission）
    for (PaperIntermission intermission : intermissions) {
        List<QuestionMedia> intermissionMedia = questionMediaBiz.list(
            new LambdaQueryWrapper<QuestionMedia>()
                .eq(QuestionMedia::getIntermissionId, intermission.getId())
                .eq(QuestionMedia::getMediaType, 9)
        );
        packMediaFiles(zos, intermissionMedia, "intermission/");
    }
    
    // 5. 题目音频和讲解（media_type=4,5,6，关联到 question）
    for (Integer questionId : questionIds) {
        List<QuestionMedia> questionMedia = questionMediaBiz.list(
            new LambdaQueryWrapper<QuestionMedia>()
                .eq(QuestionMedia::getQuestionId, questionId)
                .in(QuestionMedia::getMediaType, Arrays.asList(4, 5, 6))
        );
        
        // 分别处理题目音频和讲解音频
        for (QuestionMedia media : questionMedia) {
            String basePath;
            if (media.getMediaType() == 4) {
                basePath = "questions/";
            } else if (media.getMediaType() == 5) {
                basePath = "explanations/";
            } else {
                basePath = "questions/";
            }
            packMediaFiles(zos, Arrays.asList(media), basePath);
        }
    }
    
    // 6. 选项音频（media_type=2，关联到 question_answer）
    for (Integer questionId : questionIds) {
        // 获取题目的所有答案选项
        List<QuestionAnswer> answers = questionAnswerBiz.list(
            new LambdaQueryWrapper<QuestionAnswer>()
                .eq(QuestionAnswer::getQuestionId, questionId)
        );
        
        for (QuestionAnswer answer : answers) {
            List<QuestionMedia> optionMedia = questionMediaBiz.list(
                new LambdaQueryWrapper<QuestionMedia>()
                    .eq(QuestionMedia::getOptionId, answer.getId())
                    .eq(QuestionMedia::getMediaType, 2)
            );
            packMediaFiles(zos, optionMedia, "options/");
        }
    }
}

/**
 * 打包媒体文件到ZIP
 */
private void packMediaFiles(ZipOutputStream zos, List<QuestionMedia> mediaList, String basePath) throws Exception {
    for (QuestionMedia media : mediaList) {
        if (StringUtils.isEmpty(media.getMediaUrl()) && StringUtils.isEmpty(media.getMediaPath())) {
            continue;
        }
        
        try {
            // 从OSS下载文件
            String objectKey = StringUtils.isNotEmpty(media.getMediaPath()) 
                ? media.getMediaPath() 
                : ossUtil.getObjectKey(media.getMediaUrl());
            
            byte[] fileBytes = ossUtil.downloadFileToBytes(objectKey);
            
            // 构建ZIP中的文件路径
            String zipPath = basePath + media.getMediaName();
            
            // 添加到ZIP
            ZipEntry entry = new ZipEntry(zipPath);
            entry.setSize(fileBytes.length);
            entry.setTime(System.currentTimeMillis());
            zos.putNextEntry(entry);
            zos.write(fileBytes, 0, fileBytes.length);
            zos.closeEntry();
            
        } catch (Exception e) {
            log.error("下载媒体文件失败：mediaId={}, url={}", media.getId(), media.getMediaUrl(), e);
        }
    }
}
```

```java
public byte[] generatePaperPackage(Integer paperId) throws ServiceException {
    // 1. 获取试卷基本信息
    Paper paper = paperBiz.getById(paperId);
    
    // 2. 获取卷别列表
    List<PaperVolume> volumes = paperVolumeBiz.list(
        new LambdaQueryWrapper<PaperVolume>()
            .eq(PaperVolume::getPaperId, paperId)
            .orderByAsc(PaperVolume::getVolumeOrder)
    );
    
    // 3. 获取大题列表
    List<PaperSection> sections = paperSectionBiz.list(
        new LambdaQueryWrapper<PaperSection>()
            .eq(PaperSection::getPaperId, paperId)
            .orderByAsc(PaperSection::getVolumeCode)
            .orderByAsc(PaperSection::getSectionOrder)
    );
    
    // 4. 获取中场配置
    List<PaperIntermission> intermissions = paperIntermissionBiz.list(
        new LambdaQueryWrapper<PaperIntermission>()
            .eq(PaperIntermission::getPaperId, paperId)
    );
    
    // 5. 获取题目关联列表（按 section_id 和 section_order 排序）
    List<PaperQuestion> paperQuestions = paperQuestionBiz.list(
        new LambdaQueryWrapper<PaperQuestion>()
            .eq(PaperQuestion::getPaperId, paperId)
            .orderByAsc(PaperQuestion::getSectionId)
            .orderByAsc(PaperQuestion::getSectionOrder)
    );
    List<Integer> questionIds = paperQuestions.stream()
        .map(PaperQuestion::getQuestionId)
        .collect(Collectors.toList());
    
    // 6. 生成ZIP包
    ByteArrayOutputStream baos = new ByteArrayOutputStream();
    ZipOutputStream zos = new ZipOutputStream(baos, StandardCharsets.UTF_8);
    
    try {
        // 生成 manifest.json
        Map<String, Object> manifest = buildManifest(paper, volumes, sections, intermissions, paperQuestions);
        addJsonToZip(zos, "manifest.json", manifest);
        
        // 生成 questions.json
        List<Map<String, Object>> questionsData = buildQuestionsData(questionIds, paperQuestions);
        addJsonToZip(zos, "questions.json", questionsData);
        
        // 下载并打包媒体文件
        downloadAndPackMediaFiles(zos, paper, volumes, sections, intermissions, questionIds);
        
        zos.finish();
        zos.flush();
        
    } finally {
        zos.close();
    }
    
    return baos.toByteArray();
}
```
